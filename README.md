# Point Well Stated - Marketing Site

Simple, clean homepage for Point Well Stated.

## Quick Start

1. Open `index.html` in your browser to preview locally
2. Edit the content directly in `index.html`
3. Replace placeholder images with your product screenshots

## Deployment Options

### Option 1: Netlify
- Drag and drop the folder to [netlify.com/drop](https://app.netlify.com/drop)
- Or connect to GitHub and auto-deploy

### Option 2: Vercel
- Run `vercel` in this directory
- Or connect to GitHub

### Option 3: Cloudflare Pages
- Connect your GitHub repo at [pages.cloudflare.com](https://pages.cloudflare.com)

### Option 4: Any Static Host
- This is just HTML/CSS/JS - works on any web server
- Upload to S3, GitHub Pages, etc.

## Customization

- **Colors**: Edit CSS variables in the `:root` section
- **Content**: Edit text directly in HTML
- **Images**: Add your screenshots/images
- **Sections**: Add/remove sections as needed
